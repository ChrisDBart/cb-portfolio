Case study

CloudCasa

Summary

A cloud storage app that assists and educates consumers to simplify and streamline the home purchase process.

Created a marketing website and several use case scenarios for both desktop and mobile Apple iOS formats.

Problem

The client stated they would like to create a cloud storage app that features:

  • Save content
    • 
  • Organize content (ie. categories, tags, groups, folders)
    • 
  • Create content (ie. notes, documents)
    •  
  • Upload files from a computer or mobile device
    •  
  • Share files and folders
    • 
Solution

The app solves client problem of creating a feature rich cloud storage app through an easy-to-use interface.

The app solves user problems of data security, streamlined communication, paperless documents and informative tutorials for the home purchase process.

Set aside personal bias

Follow the data and goals of the client and users.
